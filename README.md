# AI Lab Assignments

This repository contains my Artificial Intelligence laboratory assignments, organized by assignment number. Each folder includes the Python programs and supporting files used for that experiment or exercise.

## Contents

- `Assignment-01` to `Assignment-14`: Python solutions for the lab assignments
- `PDF/` and `DOCX/`: submitted documentation files

## Requirements

- Python 3.x

Most scripts are standalone and do not require external libraries beyond the standard library.

## How To Run

Run any assignment script directly with Python from the repository root:

```bash
python Assignment-01/1.py
```

If a script depends on a helper module in the same folder or the repository root, run it from the project root so imports resolve correctly.

## Notes

- Some assignments include more than one script, usually named `1.py`, `2.py`, or `3.py`.
- A few folders also include problem statements or output files for submission.
- The `PDF/` and `DOCX/` folders contain the formatted assignment submissions.
